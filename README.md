# Cricx Database

A full-stack web application for cricket data management with data ingestion and statistics dashboard.

## Features

- **Data Ingestion Page**: Upload cricket statistics via form or CSV file
- **Statistics Dashboard**: View, filter, and analyze cricket data with visualizations

## Project Structure

```
├── frontend/          # React.js frontend application
├── backend/           # Node.js/Express.js backend API
└── database/          # Database schemas and migrations
```

## Getting Started

See individual README files in `frontend/` and `backend/` directories for setup instructions.

## Tech Stack

- **Frontend**: React.js + TypeScript + Tailwind CSS
- **Backend**: Node.js + Express.js
- **Database**: SQLite
- **UI**: Responsive design with Tailwind CSS
